## Getting Started

### Prerequisites

- NodeJS
[Visit the official Node.js website](https://nodejs.org/en/download/)

- Vite
```sh
npm install -g vite
```

### Installation

1. Download the repository

```sh
git clone 
```

2. Install NPM packages

```sh
cd ./front-mentor-product-recipe-page-challenge
npm install 
```

3. Run the app

```sh
npm run dev
```
